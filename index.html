Covex Studios Loading Screen Documentation

Welcome to the step-by-step guide for customizing the Covex FiveM loading screen.

🔧 Installation

  1. Place the folder in your resources directory.
  2. Add start covex_loading to your server.cfg.
  3. Ensure your folder includes the files listed in fxmanifest.lua.

🏷️ Change Server Name

  1. Open public/html/index.html.
  2. Press CTRL + F and search for Covex Studios.
  3. Edit the line: 
    <span>Welcome to Covex Studios!</span>
  4. Replace the paragraph below with your server description.

👥 Edit Staff

  1. Open script.js and search for staffList.
  2. Add staff using this format: 
    {
  name: \"John Doe\",
  role: \"Admin\",
  image: \"../../assets/staff/john.png\"
},
  3. Place the image into assets/staff/.
  4. Use square PNG/JPG images (e.g., 256x256).

🎵 Music Setup

  1. Open script.js and search musicTracks.
  2. Use this format for new tracks: 
    {
  src: \"../../assets/music/song.mp3\",
  title: \"Track Name\",
  artist: \"Artist\",
  image: \"../../assets/music_art/art.png\"
},
  3. Place files in assets/music/ and assets/music_art/.

📜 Server Rules

  1. Open script.js and find serverRules.
  2. Edit the rules like this: 
    "No exploiting.",
"Respect all players.",

🌐 Social Links

  1. Open index.html and search discord.gg.
  2. Update the href attributes with your URLs.

📺 Background Video

  1. Replace assets/background.mp4.
  2. Or update this line in index.html: 
    <source src=\"../../assets/YOUR_VIDEO.mp4\">

🖼 Logo

  1. Replace assets/logo.png with your logo.
  2. Update path in index.html if needed.

💡 Dev Tips

  • Use CTRL + F to find config sections quickly.
  • Search for Covex, staffList, musicTracks, or serverRules.
  • Restart your server or refresh to apply changes.
  • Always verify asset paths are correct.
